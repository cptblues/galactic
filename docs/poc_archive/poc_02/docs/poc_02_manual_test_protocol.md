# POC 0.2 Manual Test Protocol

## Densite

1. Lancer `cargo run --release`.
2. Verifier 500 systemes par defaut.
3. Zoomer/dezoomer et tourner la galaxie.
4. Tester `1`, `2`, `3`, `4` pour les presets de filtres.

## Navigation

1. Selectionner un systeme dense.
2. Tester `Tab` si le panneau ambigu apparait.
3. Utiliser `P` pour basculer en projection aplatie.
4. Utiliser `Home`, puis `Alt+Gauche` et `Alt+Droite`.

## Recherche

1. Appuyer sur `/`.
2. Chercher `oceanique`, `habitable`, un nom de systeme ou un empire.
3. Valider avec `Entree`.

## Mission

1. Appuyer sur `M`.
2. Chercher `oceanique`.
3. Ouvrir un systeme candidat, selectionner une planete.
4. Appuyer sur `Y`.
5. Verifier succes ou raison d'echec.

## Presets

- `Ctrl+1` : 100 systemes.
- `Ctrl+2` : 500 systemes.
- `Ctrl+3` : 1000 systemes diagnostic.
- `F4` : Low/Medium/High.
